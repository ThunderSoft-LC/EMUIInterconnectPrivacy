Daftar Pengumpulan Informasi Pribadi

ThunderSoft (selanjutnya disebut kami) akan secara ketat mematuhi persyaratan hukum dan peraturan negara, mengintegrasikan legalitas dan kepatuhan ke dalam desain produk dan proses bisnis, serta menggunakan teknologi keamanan canggih di industri untuk menjamin keamanan data. Pada saat yang sama, kami sangat memperhatikan perlindungan informasi pribadi dan privasi Anda, berdedikasi untuk mempertahankan hak privasi setiap orang, untuk itu kami berpegang pada 5 prinsip dasar perlindungan privasi:

●  Minimalisasi data: Bersikeras hanya mengumpulkan data pribadi yang diperlukan untuk mewujudkan fungsi bisnis, demi melayani kebutuhan Anda.

●  Pemrosesan di sisi perangkat: Bersikeras menyelesaikan pemrosesan dan analisis data pribadi pada perangkat pengguna semaksimal mungkin.

●  Transparan dan terkendali: Saat mengumpulkan data pribadi, beri tahu pengguna dengan jelas dan tegas, serta pastikan pengguna mengetahui bagaimana data digunakan, dan bagaimana cara keluar.

●  Perlindungan identitas: Menggunakan teknologi peningkatan privasi, menyembunyikan identitas pengguna saat data meninggalkan perangkat pengguna.

Penemuan dan Koneksi Perangkat Agar perangkat Anda dapat saling terhubung, kami dapat mengumpulkan pengidentifikasi unik perangkat Anda, jenis perangkat, model perangkat, nama perangkat, alamat Bluetooth, alamat Wi-Fi, SSID Wi-Fi, serta alamat IP untuk menemukan perangkat di sekitar Anda yang tersedia untuk dipasangkan dan dihubungkan, membangun tautan komunikasi antar perangkat, guna mewujudkan aliran konten dan kolaborasi fungsi antar perangkat untuk Anda. Informasi Perangkat Pengumpulan oleh sistem Pengidentifikasi unik perangkat, jenis perangkat, model perangkat, nama perangkat, alamat Bluetooth, alamat Wi-Fi, SSID Wi-Fi, serta alamat IP
Transfer File Bersama Untuk menyediakan "Transfer File Bersama" bagi Anda, melalui fungsi ini Anda dapat mewujudkan transfer dan aliran file antar perangkat. Untuk itu kami akan mengumpulkan pengidentifikasi perangkat Anda (ID perangkat), nama perangkat, jenis perangkat, informasi file (nama file serta konten file), informasi kontak yang terkandung dalam file (termasuk nama, ulang tahun, nomor telepon pribadi, alamat email, alamat, jabatan serta tempat kerja), nama Wi-Fi, dan lain-lain.
