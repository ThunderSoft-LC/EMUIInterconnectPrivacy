Pahayag tungkol sa EMUI Interconnect at Privacy

Petsa ng Pag-update: 23 Nobyembre 2025

Ang “EMUI Interconnect” ay isang cross-platform na aplikasyon para sa paglilipat ng malalaking file na ibinibigay ng Thundersoft Software Co., Ltd. (rehistradong address: Thundersoft Building, No. 9, Qinghua Donglu, Haidian District, Beijing) (mula rito ay “kami” o “Thundersoft”). Binibigyang-halaga ng Thundersoft ang proteksyon ng iyong personal na impormasyon at privacy. Magbibigay kami ng naaangkop na mga hakbang sa seguridad para sa iyong personal na impormasyon alinsunod sa mga kinakailangan ng batas at mga matatag na pamantayan ng seguridad sa industriya.

Buod

Maaari mong mabilis na maunawaan ang mga layunin, pamamaraan, at saklaw ng pangongolekta at paggamit ng personal na impormasyon ng “EMUI Interconnect” sa pamamagitan ng pagbisita sa “Listahan ng Koleksyon ng Personal na Impormasyon”.

• Upang makapagbigay ng mas mahusay na serbisyo ng EMUI Interconnect kailangan naming kolektahin ang: identifier ng device, modelo ng device, pangalan ng device, bersyon ng system, impormasyon sa network (lokal na IP address, IP address ng gateway).

• Upang maprotektahan ang mga karapatan ng Thundersoft, mo, o iba pang user ng Thundersoft, magsasagawa kami ng kontrol sa panganib sa nakolektang impormasyon sa itaas upang maiwasan ang pandaraya at iba pang ilegal na aktibidad at mabawasan ang panganib sa kredito.

Ang iyong paglilipat ng file ay maaaring may kasamang sensitibong personal na impormasyon (hal., album, contact, atbp.). Makukuha lang namin ang naturang data pagkatapos ng iyong tahasang pahintulot sa loob ng aplikasyon. Maaari mong wakasan ang koleksyon at pagproseso ng mga nabanggit na data o function sa pamamagitan ng pag-uninstall ng aplikasyong ito. Ang iyong mga nabanggit na data ay iimbak sa teritoryo ng Republikang Bayan ng Tsina.

1. Paano namin kinokolekta at ginagamit ang iyong personal na impormasyon

(a) Impormasyon ng iyong device: Kapag ginamit mo ang serbisyong ito sa isang partikular na device, kinokolekta namin mula sa device na iyon ang partikular na impormasyon, kabilang ang identifier ng device, modelo ng device, pangalan ng device, bersyon ng system, impormasyon sa network (lokal na IP address, IP address ng gateway). Ginagamit ito upang makilala ang bersyon, mabilis na masuri ang mga error, mag-troubleshoot, magbigay sa iyo ng pagtatasa ng compatibility, napapanahong i-optimize ang produkto, at magbigay ng mga abiso sa pag-update ng bersyon.

(b) Impormasyon ng iyong log: Sa iyong paggamit ng serbisyong ito, kokolektahin namin ang impormasyon ng log ng serbisyong ito (kabilang ang tagal ng paggamit, dami ng nailipat na data, uri ng data, katayuan ng operasyon, katayuan ng koneksyon, at iba pang impormasyon ng log ng pag-uugali). Ang nakolektang impormasyon ng log mula sa iyo ay gagamitin lamang para sa pagbibigay ng pangunahing serbisyo, pagsusuri ng mga problema at error na naganap o maaaring mangyari sa panahon ng paggamit ng serbisyo, at hindi ikokonekta sa iyong pagkakakilanlan.

(c) Impormasyon ng lokasyon: Kapag kumokonekta ang device, kinakailangang umasa sa iyong impormasyon ng lokasyon upang kumonekta at kunin ang device, upang makumpleto ang kinakailangang functional na serbisyo. Ang impormasyon ng lokasyon ay iniimbak lamang sa lokal na cache, hindi ina-upload sa server, at hindi ginagamit para sa iba pang hindi awtorisadong layunin.

(d) Data ng paglilipat ng user: Kapag gumagamit ka ng mga function tulad ng paglilipat ng file, kailangan naming i-cache ang mga larawan, video, teksto, contact, at iba pang nilalamang pipiliin mo, pati na rin ang katayuan ng pahintulot ng application o module ng serbisyo na iyong inililipat, upang matulungan kang makumpleto ang paglilipat ng nabanggit na nilalaman ng data. Ang kaugnay na data ay ginagamit lamang upang makumpleto ang paglilipat; hindi kami nagsasagawa ng pagkilala sa nilalaman, pagkilala sa intensyon, at hindi namin ginagamit ang mga ito para sa anumang iba pang layunin.

Sa pamamagitan ng EMUI Interconnect ay bibigyan ka namin ng mga sumusunod na function ng negosyo. Sa iyong paggamit ng mga nauugnay na function ng negosyo, ipoproseso namin ang impormasyong kinakailangan upang maibigay ang mga function na ito upang matupad ang aming mga kontraktwal na obligasyon. Kung hindi mo ibibigay ang kaugnay na impormasyon, maaaring makaapekto ito sa iyong paggamit ng mga kaugnay na function ng application na ito.

1.1 Paglilipat ng file

(1) Kapag ginagamit mo ang function ng paglilipat ng file, kailangan naming kolektahin ang iyong identifier ng device (UUID), modelo ng device, at address ng Bluetooth upang makilala at ipares sa iyong iba pang device.

(2) Maaari mong gamitin ang function na ito upang magtatag ng koneksyon sa ibang device upang makatanggap mula sa device na iyon ng mga contact, file, larawan, video, at iba pang sinusuportahang nilalaman. Upang makamit ang nabanggit na layunin, kailangan naming basahin ang napili mong nabanggit na nilalaman ng user. Ang nilalamang ito ay awtomatikong kokolektahin at ipoproseso namin pagkatapos ng iyong pahintulot kapag ginagamit mo ang function ng paglilipat ng file.

1.2 Tulong at feedback

Ang function na “Tulong at feedback” ay nagbibigay sa iyo ng mga solusyon sa mga madalas itanong. Kapag ginagamit mo ang function na “Tulong at feedback”, kinokolekta namin ang impormasyon ng hardware ng iyong device (modelo ng device, uri ng device) at paglalarawan ng problema upang magbigay ng kaukulang resulta at sagutin ang iyong mga tanong.

2. Pagtawag sa mga pahintulot ng device

Pahintulot sa lokasyon: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang matukoy kung ikaw ay nasa parehong lokal na network. Kung hindi mo kailangan ang naturang serbisyo, maaari mong i-off ang pahintulot na ito anumang oras.

Pahintulot sa lokal na network at wireless data: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang ikonekta ang mga device at maglipat ng mga file sa parehong lokal na network.

Pahintulot sa Bluetooth: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang gamitin ang Bluetooth sa paghahanap ng mga kalapit na device.

Pahintulot sa real-time na aktibidad: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang ipakita ang progreso ng paglilipat sa pamamagitan ng dynamic na isla kapag ang application ay tumatakbo sa background. Kung hindi mo kailangan ang naturang serbisyo, maaari mong i-off ang pahintulot na ito anumang oras.

Pahintulot sa mga contact: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang basahin o isulat ang impormasyon ng contact. Kung hindi mo kailangan ang naturang serbisyo, maaari mong i-off ang pahintulot na ito anumang oras.

Pahintulot sa mga larawan: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang basahin o isulat ang impormasyon ng larawan at video. Kung hindi mo kailangan ang naturang serbisyo, maaari mong i-off ang pahintulot na ito anumang oras.

Pahintulot sa camera: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang pahintulot na ito upang gamitin ang camera sa pagkuha ng mga larawan para sa feedback. Kung hindi mo kailangan ang naturang serbisyo, maaari mong i-off ang pahintulot na ito anumang oras.

Pahintulot sa pag-iimbak ng dokumento: Kapag ginagamit mo ang function ng paglilipat ng file, maaari mong paganahin ang mga pahintulot para sa mga larawan, camera, contact, atbp., upang basahin o isulat ang mga larawan, video, impormasyon ng file na ginagamit ng application. Kung hindi mo kailangan ang naturang serbisyo, maaari mong i-off ang pahintulot na ito anumang oras.

Pagkatapos makuha ang iyong pahintulot, ibibigay namin sa iyo ang serbisyo ng paglilipat ng file at ipoproseso ang kaugnay na impormasyon.

3. Proteksyon ng mga menor de edad

Binibigyang-halaga namin ang proteksyon ng personal na impormasyon ng mga menor de edad. Mahigpit na susundin ng Thundersoft ang mga batas at regulasyon ng estado sa pagbibigay ng serbisyo sa mga menor de edad at pagprotekta sa kanila. Kung ikaw ay menor de edad, kailangan mo ng pahintulot ng iyong mga magulang o iba pang tagapag-alaga upang magamit ang application na ito at sumang-ayon sa mga nauugnay na tuntunin ng serbisyo. Ang mga magulang o iba pang tagapag-alaga ay dapat ding gumawa ng naaangkop na mga hakbang sa pag-iwas upang maprotektahan ang mga menor de edad, kabilang ang pangangasiwa sa kanilang paggamit ng application na ito.

Lalo na, kung ikaw ay isang bata (menor de edad na wala pang labing-apat na taong gulang), bago gamitin ang aming mga serbisyo, tiyaking makuha ang pahintulot o gabay ng iyong mga magulang o iba pang tagapag-alaga, pagkatapos ay gamitin ang aming mga serbisyo o magbigay ng impormasyon sa amin. Kung ikaw ay magulang o tagapag-alaga ng isang bata, tiyaking ang bata sa ilalim ng iyong pangangalaga ay gumagamit ng aming mga serbisyo at nagbibigay sa amin ng impormasyon sa iyong pahintulot o sa ilalim ng iyong gabay.

4. Pagbabahagi sa ikatlong partido

Upang mabigyan ka ng ilang partikular na function/serbisyo, halimbawa upang ipakita ang impormasyon ng file, impormasyon ng contact sa Huawei Share, kailangan naming ibigay ang iyong personal na impormasyon sa mga ikatlong partido. Upang matiyak ang seguridad ng iyong data at privacy, pumirma kami ng mga kasunduan sa proteksyon ng seguridad ng data sa mga kasosyo, na tumutukoy sa kanilang responsibilidad, obligasyon, at kinakailangan para sa proteksyon ng iyong data. Para sa mga detalye, sumangguni sa “Listahan ng Pagbabahagi ng Personal na Impormasyon”.

4.1 Pagbawi ng pahintulot

Kung kailangan mong bawiin ang isang pahintulot, maaari kang pumunta sa “Mga Setting” > “Pamamahala ng Pahintulot” at i-off ang kaukulang switch upang bawiin ang iyong pahintulot.

5. Lugar ng pag-iimbak ng impormasyon

5.1 Lugar ng pag-iimbak

(1) Ang impormasyon ng iyong device, impormasyon ng lokasyon, at data ng paglilipat ng user ay iimbak sa lokal na storage ng device.

(2) Ang iyong impormasyon ng log at nilalaman ng feedback ay iimbak sa teritoryo ng Republikang Bayan ng Tsina.

5.2 Panahon ng pag-iimbak

Iniimbak lang namin ang iyong personal na impormasyon sa panahong kinakailangan upang makamit ang mga layuning nakasaad sa pahayag na ito. Pagkatapos mong kanselahin ang serbisyo, aktibong tanggalin, o mag-expire ang panahon ng pag-iimbak, agad naming tatanggalin o ipalilihim ang iyong personal na impormasyon, maliban kung ang mga batas at regulasyon ay nagtatakda ng iba (hal.: ang “Cybersecurity Law” ay nangangailangan ng pag-iingat ng mga log ng network sa loob ng 6 na buwan). Bukod pa rito, kapag ang aming produkto o serbisyo ay tumigil sa operasyon, ipaalam namin sa iyo sa pamamagitan ng push notification, anunsyo, atbp.
