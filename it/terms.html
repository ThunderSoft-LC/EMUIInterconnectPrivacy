Questo servizio è progettato per fornirti la funzione di trasferimento file tra dispositivi, aiutandoti a soddisfare le tue esigenze di trasferimento rapido di contenuti. Nota: Le funzioni supportate su diversi sistemi e modelli possono differire leggermente; fare riferimento alla versione effettiva del software che stai utilizzando per le funzioni supportate o aggiunte. Le funzioni principali di questo servizio sono:

(1) Impostazioni delle informazioni personali: puoi impostare tu stesso il nome che utilizzerai quando usi questo servizio.

(2) Trasferimento file reciproco: attraverso questo servizio puoi stabilire una connessione con un altro dispositivo affidabile (come telefono/computer) per inviare all'altra parte o ricevere dall'altra parte contatti, file, immagini, video e altri contenuti supportati da questo servizio (collettivamente "contenuto utente").
