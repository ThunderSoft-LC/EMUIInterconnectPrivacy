EMUI Interconnect erabiltzaile-zerbitzuaren hitzarmena

Eguneratze-data: 2025eko azaroaren 23a

Erabiltzaile errespetatua:

EMUI Interconnect zerbitzua (aurrerantzean “zerbitzu hau”) Thundersoft Software Co., Ltd. enpresak (aurrerantzean “gu” edo “Thundersoft”) kudeatzen du. Erabiltzaile-zerbitzuaren hitzarmen hau zuk (erabiltzaileak) Thundersoft Technology Co., Ltd. eta bere lizentzia-emaileekin sinatzen duzun akordio juridiko balioduna da. Hitzarmen honen bidez, Thundersoft-ek zerbitzu hau nola eskaintzen dizun, zerbitzu hau erabiltzean jarraitu behar dituzun arauak, Thundersoft-ek zerbitzu hau eta hari lotutako zerbitzuak emateko deklarazioak eta bestelako informazio garrantzitsua ezagutuko dituzu. Mesedez, irakurri hitzarmen hau arretaz, bereziki aipatuta dauden eztabaiden jurisdikzioari, erantzukizunaren ukoari eta informazioa biltzeari buruzko klausulak. 18 urte beteta izan behar duzu, edo zure eskualdeko lege-adina izan, zerbitzu honetara sartzeko eta erabiltzeko. 18 urtetik beherakoa bazara, zure gurasoen edo legezko tutorearen baimena behar duzu zerbitzu hau erabiltzeko. Adingabeek hitzarmen hau irakurri behar dute legezko tutorearekin batera. Softwarea deskargatzea, instalatzea eta zerbitzu hau erabiltzea aukeratzen baduzu, hitzarmen honen klausula guztiak onartu dituzula eta Thundersoft-ekin legez loteslea den akordioa sinatu duzula ulertuko da.

1. Zerbitzu honi buruz

Zerbitzu honek gailuetako fitxategien transferentzia funtzioa eskaintzen du, edukia azkar transferitzen laguntzeko.Oharra: sistema ezberdinetan eta modelo ezberdinetan egokitutako funtzioak apur bat desberdinak izan daitezke; onartzen diren edo gehitzen diren funtzio zehatzak zuk benetan erabiltzen duzun softwarearen bertsioaren araberakoak dira. Zerbitzu honen funtzio nagusiak hauek dira:

(1) Informazio pertsonalaren ezarpenak: zuk zeuk ezar dezakezu zerbitzu hau erabiltzean erabiliko duzun izena.

(2) Fitxategien elkar-transferentzia: zerbitzu honen bidez beste konfiantzazko gailu batekin (adibidez, telefono/ordenagailu) konexioa sor dezakezu, gailu horretako kontaktuak, fitxategiak, irudiak, bideoak eta zerbitzu honek onartzen dituen beste eduki batzuk bidali/jaso (elkarrekin “erabiltzaile-edukia” deituak).

(3) Zerbitzu honen bidez hirugarrenen zerbitzu bat erabiltzea aukeratzen baduzu, mesedez, ikusi hirugarrenen zerbitzu-hornitzaileak aurkeztutako dagozkion hitzarmenen edukia. Hirugarrenen zerbitzuak erabiltzean sortutako gatazkak, zuk zeuk jarri behar duzu harremanetan hirugarrenekin konpontzeko.

2. Softwarearen lizentziari buruz

(1) Lizentzia ematea. Thundersoft-ek ondoko klausulen arabera, lizentzia mugatu, ez-esklusibo eta ez-transferigarria ematen dizu. Software hau eta dagozkion dokumentuak erabiltzeko baimena ematen du, baina ez duzu lizentzia azpilizentziatzeko eskubiderik.

(2) Lizentziaren mugak

    (a) Software hau gailu mugikor batean deskarga, instalatu eta erabil dezakezu erabilera komertzialik gabe, Thundersoft-ek argi eta garbik baimendu ezean edo dagokion legeak hitzarmen honetako mugak debekatzen ez baditu. Bestela, ezin duzu: (i) beste pertsonei software hau erabiltzen utzi; (ii) softwarea edo dokumentuak aldatu, itzuli, ingeniaritza alderantziz egin, deskonpilatu, desmuntatu, edo software edo dokumentuetan oinarritutako produktu eratorriak sortu; (iii) softwarea edo dokumentuak kopiatu (konponketa/ordezko kopiak izan ezik, baldin eta kopiak ez badira ordenagailu batean instalatu edo erabiltzen, eta konponketa/ordezkapenaren ondoren berehala ezabatzen/suntsitzen badira. Softwarearen egiten duzun beste edozein babeskopia hitzarmen honen hausteatzat hartuko da); (iv) softwarea edo dokumentuak alokatu, transferitu, azpilizentziatu, edo software edo dokumentuen gaineko eskubideak besteei transferitu; (v) softwarean edo dokumentuetan dauden jabetza-oharrak edo etiketak kendu; (vi) software honi beste software bat gehitu edo beste software bat sistema eragile edo software honekin bateratu.

    (b) Softwarea eta dokumentuak lege garrantzitsu guztiei jarraituz soilik erabiltzeko baimena ematen duzu.

    (c) Ezin duzu zerbitzu hau legez kanpoko jardueretan erabili, ezta baimenik gabe zerbitzu honen ahuleziak eskaneatzen saiatu edo zerbitzu honen funtzionamendu normala oztopatu, birus kaltegarriak zabaldu edo sareko informazio-zerbitzu normalak oztopatzen dituzten beste jokabide batzuk izan.

3. Jabetza intelektualaren adierazpena

Software honen eta dokumentuen jabetza eta jabetza intelektuala Thundersoft-enak dira (hornitzaile/lizentzia-emaile guztiak barne). Software honen bidez atzitutako edukiaren jabetza eta jabetza intelektuala (softwarearen komunikabideen aurkezpen-fitxategietan dagoen edukia barne) dagokion edukiaren jabeena izaten jarraitzen du, eta dagokion copyright-ak edo beste lege batzuek babes dezakete. Lizentzia honek ez dizu ematen horrelako edukiaren gaineko eskubiderik.

4. Softwarearen instalazioari eta eguneratzeari buruz

Software hau deskargatu eta instalatu behar duzu deskarga-kanal ofizialetatik, ez deskargatu ez-ofizialeko webguneetatik, birusen edo software kaltegarrien erasoen jasan ez zaitezen. Aldi berean, zure terminal-gailuaren modeloarekin bat datorren software-bertsioa aukeratu behar duzu, edo zeuk bertsio-eguneratzea egin. Bestela, software-bertsioa eta gailu-modeloa bat ez datozelako sortutako software-erabilera arazoak, funtzioen falta, gailu-lotura arazoak, etab. zure kargura izango dira.Softwarea deskargatzean eta eguneratzean datu-trafikoa edo informazio-kostuak sor daitezke (dagozkion kostuak telekomunikazio-operadoreak kobratzen ditu), zuk zeuk ordaindu beharko dituzu.

5. Erabileraren konpromisoa eta bermea

Hitzarmen honen klausulen arabera, zure izena ezar dezakezu zerbitzu honen bidez, eta fitxategi-datuak transferitu, etab. Baina hitzeman behar duzu zerbitzu hau modu legezko eta arautegikoan erabiliko duzula. Aldi berean, bermatu behar duzu zerbitzu hau (ezarpen pertsonalizatuen, elkar-transferentziaren, etab. bidez) herrialdeko legeak eta arauak urratzen dituen edozein informazio sustatzeko ez duzula erabiliko, besteak beste, baina ez mugatuta, honako informazioa:

(a) Konstituzioak ezarritako oinarrizko printzipioen aurka doana;

(b) Estatuaren segurtasuna arriskuan jartzen duena, estatuaren sekretuak zabaltzen dituena, estatuaren boterea eraisten duena, estatuaren bateratasuna hausten duena;

(c) Estatuaren ohore eta interesak kaltetzen dituena;

(d) Nazio-gorrotoa, nazio-diskriminazioa sustatzen duena, nazio-batasuna hausten duena;

(e) Estatuaren erlijio-politika urratzen duena, erlijio-sektak eta sineskeri feudalak sustatzen dituena;

(f) Zurrumurruak zabaltzen dituena, gizarte-ordena nahasten duena, gizarte-egonkortasuna ahultzen duena;

(g) Lizunkeria, pornografia, jokoak, bortizkeria, hilketak, terrorismoa zabaltzen dituena edo delitura sustatzen duena;

(h) Beste batzuk iraintzen edo difamatzen dituena, besteen legezko eskubideak eta interesak urratzen dituena;

(i) Besteen jabetza intelektuala, sekretu komertzialak edo bestelako legezko eskubideak urratzen dituena;

(j) Fede onaren, gizarte-ordena publikoaren, publiko-moralaren printzipioak hautsi ditzakeen edozein jokabide edo legeek debekatutako beste edozein.

Klausula honen baldintzak haustearen ondoriozko edozein ondorio, besteak beste, baina ez mugatuta, erantzukizun juridikoa hartzea eta kaltetutako subjektuen galerak konpentsatzea, zure kargura izango dira.

6. Erantzukizunaren ukazioa eta erantzukizunaren mugapena

Zerbitzu hau zure erabilera pertsonalerako soilik da, ezin zaio hirugarren bati eman. Halaber, software, dokumentu eta eduki hau “den bezala” eskaintzen da. Thundersoft-ek ez du ematen adierazpen edo berme esplizitu edo inpliziturik, besteak beste, baina ez mugatuta, produktuaren merkaturatzeari, helburu zehatzerako egokitasunari, eta:

(1) Zerbitzu honek zure behar espezifikoei erantzungo diela; (2) zerbitzu hau etenik gabe, garaiz, modu seguruan edo akatsik gabe emango dela; (3) zerbitzua erabiltzean lortzen duzun edozein informazio zehatza edo fidagarria izango dela; edo (4) zerbitzuaren zati diren akats edo erroreak zuzenduko direla. Thundersoft-ek ez du inolako erantzukizunik hartzen erabiltzailearen edozein akats, legez kanpoko edo hitzarmen honen klausulak hausten dituen Thundersoft-en zerbitzu, errendimendu edo funtzioen erabileragatik. Thundersoft-ek eskaintzen dituen errendimendu, funtzio edo zerbitzuak erabiltzean indarrean dauden lege guztiak bete behar dituzu.

7. Hitzarmen honen etetea edo amaiera

Zerbitzu honen etetea edo amaiera honako egoerak biltzen ditu:

(a) Zuk zeuk zerbitzua amaitzea eskatzen duzu eta softwarea desinstalatzen duzu;

(b) Zuk hausten duzu edo Thundersoft-ek bere iritziz arrazoi duela uste duzu legeak edo arauak hausten dituzula edo hausteko susmoa duzula, edo hitzarmen honetan adostutako edozein edukia hausten duzula, lizentzia-mugak, zure bermeak eta konpromisoak barne;

(c) Legeen eta arauen arabera edo dagozkien gobernu-agentzien eskakizunen arabera, Thundersoft-ek zerbitzua eteten edo amaitzen du;

(d) Gorabehera-ezinezkoak edo aurreikusi gabeko arazo teknikoak gertatzen dira (besteak beste, baina ez mugatuta, hackerren erasoak, sarearen eroraldia, birusen inbasioa eta Interneten funtzionamendu normalari eragiten dioten beste egoerak);

(e) Legeen eta arauen arabera edo Thundersoft-ek bere kabuz ezin dituen arrazoiak, negozio-doikuntzak, etab. direla eta zerbitzua etetea edo amaitzea dagokion beste egoerak.

Aipatutako etete edo amaierako egoerak gertatzen badira:

(a) Thundersoft-ek eskubidea du zuri emandako erabilera-lizentzia eteteko edo amaitzeko, edo hitzarmen honen arabera bestelako neurri murriztaileak hartzeko;

(b) Hitzarmen honetan argi eta garbi edo izaeragatik hitzarmena amaitzean ere indarrean jarraitu behar duten edozein klausula, indarrean jarraituko du hitzarmena amaitu ondoren, adostutako baldintzak amaitu arte edo izaeragatik amaitu arte.

8. Aplikagarri den legea eta jurisdikzio klausulak

Zure bizilekuaren legeak kontrakoa xedatzen ez badu, lizentzia-hitzarmen hau Txinako Herri Errepublikako legeak arautuko du, eta Txinako Herri Errepublikako legeen arabera interpretatuko da, eta ez da gatazka-arauak aplikatuko. Onartzen duzu Dongguan Hiriko Bigaren Herri Auzitegiak duela jurisdikzioa eta epaiketa auzitegi horretan egingo dela, eta uko egiten diozu jurisdikzio edo epaiketa-leku horren gaineko edozein eragozpeni. Edozein arrazoirengatik, jurisdikzio eskuduneko auzitegi batek lizentzia honen edozein klausula edo klausularen zati bat betearazgarria ez dela erabakitzen badu, lizentzia-hitzarmenaren gainerako zatiak indarrean jarraituko du osorik.

9. Hitzarmenaren eguneratzea

Hitzarmen hau argitaratu edo eguneratu den egunetik aurrera indarrean sartzen da. Thundersoft-ek eta zerbitzu hau aktibatu eta erabiltzen duten erabiltzaile guztiek goiko hitzarmenaren mende daude.Thundersoft-ek eskubidea du bere egoera operatiboaren, merkatu-ingurunearen eta beste faktore batzuen arabera, zerbitzu-baldintzak edozein unetan aldatzeko (aldaketak dagozkien klausulen edukia aldatzea, gehitzea, gutxitzea barne ditu). Zerbitzu-baldintzak aldatzen direnean, Thundersoft-ek eguneratzea erakutsiko du dagozkion orrialdeetan. Baldintzen aldaketa honekin ez bazaude ados, zerbitzu hau erabiltzeari utzi diezaiokezu. Thundersoft-ek eskaintzen duen zerbitzua erabiltzen jarraitzen baduzu, klausula honen aldaketa guztiak onartu dituzula ulertuko da.

10. Nola jarri harremanetan gurekin

Informazio pertsonala babesteko sail espezializatua eta informazio pertsonala babesteko arduraduna sortu ditugu. Zalantzarik, iritzirik edo iradokizunik baduzu, bisitatu Thundersoft-ekin harremanetan jartzeko orria. Gurekin harremanetan jar zaitezke bezeroarentzako arreta-linearen bidez (+86-010-62662686).

Normalean, hiru lanegunetan erantzungo dugu hasieran, eta legeak ezarritako epean erantzungo ditugu zure galderak. Gure erantzunarekin pozik ez bazaude, batez ere informazio pertsonala tratatzeko gure ekintzek zure legezko eskubideak eta interesak kaltetzen badituzte, zure tokiko pribatutasun-babesaren erakunde arautzaileari kexa edo salaketa aurkez diezaiokezu.

11. Klausula orokorrak

(1) Hitzarmen honen izenburuak irakurketaren erraztasunerako soilik ezarri dira, eta ez dute eragiten oinarrizko testuko klausulen esanahiaren interpretazioan.

(2) Hitzarmen honetako edozein klausula, edozein arrazoirengatik, erabat baliogabetzat, partzialki baliogabetzat edo betearazgaitzat hartzen bada, hitzarmenaren gainerako klausulak indarrean jarraituko du eta loteslea izango da.

(3) Hitzarmen honetako edozein edukin ezin da interpretatu zure eta Thundersoft-en artean inolako lankidetza komertzial edo agentzia harremanik sortu denik.

(4) Legeek eta arauek baimendutako mugetan, Thundersoft-ek eskubidea du hitzarmen honen klausulak interpretatzeko.
